VibeLabs

Vibe Labs uses your AI Copilot prompting history (e.g., Cursor, Claude Code, Windsurf) to generate personalized coding exercises tailored to your style and habits. It teaches you to solve your most common, easy-to-solve prompts, saving the time you would otherwise spend waiting for each repetitive prompt, boosting the efficiency of your vibe coding and improving your programming skills.

Inspiration

Vibe coding empowers developers to move faster, unleash creativity, and focus on ideas rather than syntax. But with this convenience comes a tradeoff: many developers, especially beginners, end up skipping over core fundamentals. There is a pattern hidden in plain sight here: a developer's own AI chat history. The same "how do I center a div" conservation repeated ten times a day reveals a programmer's knowledge gaps. Our project transforms those repeated prompts into personalized mini-lessons tailored to each coder. This way, vibe coders not only overcome immediate challenges and ship faster today, but also build the foundational skills that will 10x their productivity in the future.

What it does

VibeLabs is an intelligent education platform that logs and analyzes the prompts developers type into AI coding engines such as Cursor and Windsurf. It estimates a developer's programming experience level and automatically generates personalized lesson plans to strengthen weak areas. Main features: (1) Real-time Prompt Tracking and Analysis: VibeLabs captures coding-related prompts. Using natural language analysis, it identifies skill level with Claude and pinpoints the specific concepts you repeatedly struggle with. (2) Personalized Lesson Generation: When a weak spot is identified, our platform automatically builds a custom lesson plan with coding exercises tailored to you. Your learning journey constantly adapts as your skills improve. (3) Interactive Practice and Feedback: You can solve these tailored exercises directly on the platform and receive instant, AI-driven feedback on your code, creating a powerful loop of practice and reinforcement. (4) Progress Dashboard and Leaderboard: Track your growth with visual insights on your personal dashboard. A gamified leaderboard adds a fun, competitive edge, motivating you to continuously level up your skills.

How we built it

Frontend: The UI is built as a React/Next.js web app with Tailwind CSS. Backend: We store user information in a Supabase PostgreSQL database which communicates between our frontend and keylogger. Data collection: We use a python based keylogger to get the Claude Code Prompts. As the user makes new prompt queries, the new prompts are synced to our database, and we parse through them to look for areas of repeated inefficiencies. AI-processing: VibeLabs uses Claude API to: 1) analyze prompt histories and find recurring knowledge gaps, 2) generate personalized coding exercises to address those gaps, and 3) provide instant, AI-powered feedback on user-submitted solutions.

Challenges we ran into

Data Acquisition: Overcoming the lack of a direct API to get a user's copilot prompts by developing a custom, secure keylogger to capture user data. Latency Management: Ensuring a zero-wait user experience by pre-processing copilot prompts and asynchronously generating exercises in the background, avoiding a bottleneck of performing all the API calls when the user visits our web app. AI Logic: Engineering precise prompts for the Claude API to accurately summarize user weaknesses to generate relevant, high-quality coding exercises, and evaluate user solutions to coding exercises.

Accomplishments that we're proud of

We're proud that in such a short time (24 hours), we built a system that takes a coder's natural prompts and automatically generates tailored lessons from them. It was very exciting seeing how well our prototype could intelligently recognise repeated requests patterns (i.e. like 'center a div') and turn them into bite-sized tutorials. We're proud that our project supports the vision of making vibe coding not only fast but also more efficient. Our prototype shows how we can help coders get answers in the moment while also building the confidence to solve challenges on their own. This reduces repeated prompts, streamlines the workflow, and demonstrates that vibe coding can be both a powerful tool for speed and a pathway for long-term growth.

What we learned

We learned a lot from the technical side of this project. For example, we figured out how to pre-process database updates and API queries to reduce latency and lag, which made our system run much smoother. Beyond the technical work, we also learned that copilot prompts aren't just instructions, they're signals that reveal where coders need support. Generating these personalized lessons directly from the vibe coding flow can turn those signals for support into opportunities for long-term growth and increasing their development efficiency.

What's next for our project

(1) VSCode Extension: Build a native extension to integrate directly with Copilot, Claude Code, and Cursor. It'd be more convenient to access directly in the IDE rather than having to go to a separate web app. Plus, with a built-in VSCode extension, we could get the user's prompt interactions without having to use a keylogger. (2) Enhance Coding Exercises: Adding difficulty levels and progressive hints if a user is stuck to boost learning. (3) Gamification: Add badges, points, leaderboards, streaks, or achievement levels to motivate learners, encourage consistent practice, and make exercises more engaging and fun.
